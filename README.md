# 🏃‍♂️ BODY LOG (体重記録アプリ)

ブラウザだけで動作する、シンプルでプライバシーに配慮した体重管理アプリです。
GitHub Pages を利用して、自分専用の管理ツールとしてどこからでもアクセスできます。

## ✨ 主な機能
* **体重記録:** 日付・時刻・メモと一緒に体重をクイック保存。
* **グラフ表示:** 1週間・1ヶ月・全期間の推移を可視化。
* **BMI計算:** 入力された身長に基づき、健康状態を自動判定（色分け機能付き）。
* **連続記録日数:** モチベーションを維持するためのストリーク表示。
* **データ永続化:** ブラウザの `localStorage` を使用。サーバーにデータは送信されません。
* **エクスポート:** 記録したデータをCSV形式でダウンロード可能。

## 🚀 使い方
1. 右上の **Settings**（歯車アイコン）から身長と目標体重を設定します。
2. 毎日の体重を入力して「記録する」ボタンを押すだけ！
3. スマホのブラウザ（Safari/Chrome等）で開き、「ホーム画面に追加」するとアプリ感覚で使えます。

## 🛠 テクニカルスタック
* **Frontend:** React (CDN), Tailwind CSS
* **Chart:** Recharts
* **Storage:** Browser LocalStorage
* **Deployment:** GitHub Pages

## 🔒 プライバシーについて
このアプリは「完全ローカル動作」です。入力された体重やメモが外部サーバーに送信されることはありません。データはあなたのブラウザ内のみに保存されます。
